Ni habari ya Mungu aliyekaribia. Ni simulizi la Mfalme wa mbinguni aliyeshuka chini, si kwa nguvu, si kwa kelele bali kwa kilio cha mtoto mchanga.  

> ‚ÄúNaye Neno alifanyika mwili, akakaa kwetu; nasi tukauona utukufu wake, utukufu kama wa Mwana pekee atokaye kwa Baba, amejaa neema na kweli.‚Äù  
> ‚Äî [Yohana 1:14](https://www.bible.com/bible/1588/JHN.1.AMP)

Hii ndiyo zawadi ya mwisho kabisa kwa roho inayokufa kwa uchovu wa kutafuta maana pasipo Mungu.  

Mungu hakutuma wazo wala sheria mpya. Alituma Nafsi Yake. Alikuja Mwenyewe.

Hakuja kutuonyesha njia tu. Yeye ndiye Njia. Hakuja kutupa mwanga kidogo gizani. Yeye ndiye Nuru kamili. Hakuja kutuambia tu kuhusu upendo wa Mungu. Yeye ndiye upendo huo, ukiwa umevaa mwili wa mwanadamu.  

Anachohitaji kutoka kwetu si nguvu, si haki tuliyoijenga wenyewe. Anachohitaji ni moyo unaosikia sauti Yake na kusema: ‚ÄúBwana, sema na mimi.‚Äù

// Form submission via AJAX
const form = document.getElementById('comment-form');
const thankYou = document.getElementById('thank-you-msg');

form.addEventListener('submit', async (e) => {
  e.preventDefault();

  const data = new FormData(form);

  try {
    const response = await fetch('https://formspree.io/f/xvzoqrkp', {
      method: 'POST',

    thankYou.innerText = "Oops! Something went wrong.";
    console.error(error);
  }
});
</script>
